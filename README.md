download the binary or build it urself

> ./4chan-scraper https://boards.4chan.org/wsg/thread/6045473 /home/requiem/Downloads/4ch <br />
this will download all media files from thread 6045473 from board wsg into ~/Downloads/4ch/{subject}
(if subject isnt set, it will use thread post unix instead)

you can also just use 
> ./4chan-scraper https://boards.4chan.org/wsg/thread/6045473 <br />
and it will dl into the cwd under {subject} or {timestamp}

no ai was used in the making of this utility !
